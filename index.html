Javascript Buttons Sandbox

Notes:

  • data-button: buynow | cart | donate | subscribe | paynow (undocumented)
  • data-type: button | form | qr
  • data-size: small | medium | large
  • data-lc: use the [language]_[country] syntax e.g. fr_FR, en_US, de_DE

Requirements:

  • data-merchant | remove the merchant ID from the URL parameters
  • data-height and data-width | undocumented features that specify pixel dimensions and are defaulted when the user specifies data-size
  • data-label | undocumented feature that allows the user to manually specify a label
  • data-locale | already implemented as data-lc
  • data-funding and data-wordmark | undocumented features that default to true and show/hide the credit card logos (bottom) and PayPal wordmark (right).

Press Tab or click in the textarea to update the button
